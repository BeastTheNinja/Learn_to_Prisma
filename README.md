# Learn to Prisma

A full-stack learning project built to explore modern web development using Prisma, Express, React, TypeScript, and MySQL.

The project started as a Prisma learning exercise and evolved into a complete authentication and user management system with role-based access control.

## Features

### Backend

* Express.js API
* Prisma ORM
* MySQL database
* JWT Authentication
* Role-based Authorization (USER / ADMIN)
* Password hashing with bcrypt
* CRUD operations
* Middleware-based route protection

### Frontend

* React + TypeScript
* React Router
* Protected Routes
* Role-based UI rendering
* Login & Registration
* User Management
* Service Layer Architecture
